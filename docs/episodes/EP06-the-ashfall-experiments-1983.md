# EP06 - "The Ashfall Experiments - 1983"

## Pillar
Incident Report

## Lore
Ashfall Experiments, unauthorized Veil interaction, facility loss

## Summary

The Directorate's greatest internal failure. A black project that went catastrophically wrong. Presented through heavily redacted recovered documents and a single surviving researcher audio log.

## Content Structure

- Project overview document - most fields redacted
- Facility layout - sections marked LOST
- Researcher audio log - calm at first, then wrong
- Repeating corridors described
- Faceless personnel described
- Recording ends mid-sentence
- Archivist annotation: *"Seventeen researchers entered. Personnel records for all seventeen were deleted the following morning."*

## Lore Introduced

- Ashfall Experiments in full
- Veil's capacity to corrupt physical space
- Directorate technology potentially derived from Ashfall materials
- First Phantom connection seeded

## First Phantom Fragment

Casualty list - personnel number **7741** appears. Still no name. Same number as EP01.

## Ends On

The researcher's final words before the recording cuts:
*"There's something in the corridor that was not part of the experiment."*

## Thumbnail Concept

Facility blueprint. One corridor highlighted in red. Label: UNACCOUNTED.
