# Creature Collection System

## Status
Core Progression System

---

# Overview

Obsidian Frequency features a creature and entity collection system.

Instead of only defeating monsters, players:
- document them
- study them
- classify them
- and collect data on them

This builds a growing paranormal archive.

---

# Core Gameplay Loop

Players encounter entities and can:
- capture evidence
- survive encounters
- extract samples
- unlock lore entries
- complete classification data

---

# Collection Motivation

Players are driven by:
- rare discoveries
- powerful entities
- lore completion
- Hunter synergy unlocks
- visual collection completion

# Entity Data Structure

Each entity contains:

- Lore
- Stats
- Behavior patterns
- Visual representation
- Environmental associations
- Capture difficulty rating
- Evolution or variant paths

---

# Collection Book

The Collection Book is a central UI system where players:
- track discovered entities
- review classified lore
- observe rarity tiers
- unlock hidden entries
- and complete research sets

---

# Retention Design

The system encourages long-term engagement through:
- rare entity hunting
- completionism
- lore discovery
- and progression unlocking

---

# Design Philosophy

Every entity should feel:
- unique
- discoverable
- mysterious
- and meaningful
