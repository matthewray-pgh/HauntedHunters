# Video 04 - The Ashfall Experiments

## Pillar
Incident Report / Directorate Intelligence

## Lore Focus
Ashfall 1983, forbidden Veil research, Directorate internal division

## Purpose
Shift tone from external horror to internal horror.
The Directorate itself did something wrong.
Introduce the idea that some factions within the organization want to open the Veil, not seal it.
First hint of The Architects faction.

## Format
- Heavily redacted internal project file
- Redaction reveals and removals mid-video
- Degraded audio log fragment from facility
- Static interference increasing as video progresses

## Content Outline
- Project overview - unauthorized breach interface experiment
- Facility description - remote, subterranean, off-record
- Activation event - what happened when the interface engaged
- Recovery team reports - repeating corridors, faceless personnel
- Casualty count - unknown, records erased
- Current status - facility lost
- First mention of recovered Ashfall materials influencing modern Directorate tech
- Closing Archivist annotation: *"Someone approved this project. That approval is not in any file I have found. Yet."*

## First Phantom Seed
Facility map shows a room with no label. Door drawn open. No notation.

## Intended Viewer Reaction
*"Someone inside the Directorate wanted this to happen. Who approved Ashfall."*
